# Ghost Story Generator

A Python-based story generator that creates unique Horror, Fantasy, Romance, Mystery, Sci‑Fi, and Adventure stories.

## Features
- Procedural character generation
- Dynamic settings and plot seeds
- Multiple genres
- Plot twists and endings
- Interactive CLI
- Save stories to text files
